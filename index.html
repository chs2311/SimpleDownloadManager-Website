Simple Download Manager

Die beste Lösung um schnell viele Dateien herunterzuladen

Jetzt herunterladen

Genervt von schlechter kostenpflichtiger Software? - Dann sind sie hier genau richtig!

Simple Download Manager ist nicht nur kostenlos, sondern bringt auch viele vorteilehafte Features
in seiner neusten Version mit sich. Profitieren sie von zeitsparenden Funktionen, die ihnen auch
ihre Arbeit vereinfachen.

Funktionen

Einfaches Teilen

Ende vieler Links?

Versenden sie viele Links als eine einfache Datei und laden sie die Links per diese Datei in das Programm.

Exportieren

Die Software noch nicht installiert?

Nutzen sie die Export-Module um einen Download auf Geräten (min. Windows 8 x64) ohne diese Software zu ermöglichen.

Einfache Linksammlung

Automatische Erkennung einer Änderung in ihrer Zwischenablage sorgt dafür, dass die Links sofort an das Programm übergeben werden.

Viele Datei-Icons

Jede Datei bekommt je nach Typ ein Icon versehen.

SimpleDownloadManager hat Icons von 1033 Dateitypen eingespeichert und kann damit Dateitypen mit Symbolen einfach erkennbar machen.
